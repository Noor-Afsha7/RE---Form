# 🏍️ Royal Enfield Registration Portal

> **Ride the legacy. Register your next adventure.**

The Royal Enfield Registration Portal is a modern, responsive web application designed for enthusiasts to register their interest in Royal Enfield motorcycles and book a test ride. Built using HTML, CSS, and JavaScript, the project focuses on creating an intuitive user experience through interactive forms, real-time validation, and dynamic UI interactions.

The application demonstrates front-end development concepts such as DOM manipulation, form validation, LocalStorage data persistence, and responsive design while maintaining a premium Royal Enfield-inspired aesthetic.

---

## ✨ Features

- 🏍️ Auto-scrolling Royal Enfield image carousel
- 📝 Interactive registration form
- ✅ Real-time form validation
- 📅 Test ride date selection
- 📍 State and motorcycle model selection
- ☑️ Radio buttons and checkboxes
- 💾 LocalStorage-based data storage
- 🎉 Animated success notifications
- 🔄 Form reset functionality
- 📱 Fully responsive layout
- ⚡ Dynamic DOM interactions
- 🎨 Premium Royal Enfield-inspired UI
